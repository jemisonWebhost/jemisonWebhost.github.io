HTML Attributes

  • <-- Previous
  • Next -->
  • All HTML elements can have attributes
  • Attributes provide additional information about an element
  • Attributes are always specifies in the start tab
  • Attributes usually come in name/value pairs like: name="value"

Examples of attributes in HTML:

  • href <a href="jemisonwebhost.github.io">This is a link!</a>
  • src <img src="resources/images/example1.png" />
  • width <img src="resources/images/example2.png" width="50%" />
  • height <img src="resources/images/example3.png" height="180" />
  • alt <img src="resources/images/example4.png" alt="This is example 4." />
  • style <p style="color:blue">This is a blue paragraph</p>
  • lang
    <html lang="en-US">
    some html
    </html>
