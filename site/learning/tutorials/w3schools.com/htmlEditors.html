Write HTML Using Notepad or TextEdit

Web pages can be created and modified by using professional HTML editors.
However, for learning HTML, it is generally agreed that a simple text editor like Notepad (PC) or TextEdit (Mac) is a good way to learn HTML. Follow the steps below to create your firse web page with Notepad or TextEdit.

  1. Open Notepad (PC)

    • Windows 8 or later:
      • Open the Start Screen. Type Notepad. Press enter.
    • Windows 7 or earlier:
      • Open Start > Programs > Accessories > Notepad

    Open TextEdit (Mac)

    • Open Finder > Applications > TextEdit

      Also, change some preferences to get the application to save files correctly. In Preferences > Format > choose "Plain Text"

      Then under "Open and Save", check the box that says "Display HTML files as HTML code instead of formatted text".

      Then open a new document to place the code.

  2. Write Some HTML

    • <!DOCTYPE html>
      <html>
      <body>

      <h1>My First Heading</h1>

      <p>My first paragraph.</p>

      </body>
      </html>


  3. Save the HTML Page

    • Save the file on your computer. Select File > Save as in the Notepad menu.
      Name the file "index.html" and set the encoding to UTF-8 (which is the preferred encoding for HTML files).

  4. View the HTML Page in Your Browser

    • Open the saved HTML file in your favorite browser (by double clicking on the file, or right-click - and choose "Open With").

More advanced options do exist for editing HTML files. Notepad++ is a lightweight, but powerful free text editor with built in support for HTML, CSS, JScript, and many other languages. It color-codes tags and strings to make your text easier to read, and there is an option to install it onto a flash drive as a portable program. This makes it ideal for programming on the go.
Get it here: https://www.notepad-plus-plus.org

Brackets is another free advanced editor. Like Notepad++, Brackets supports many programming languages, but unlike Notepad++, you can add extensions to Brackets to give it additional functionality. These extensions range from changing the colors that Brackets uses to tools that allow Git support and check your code for errors.
Get it here:brackets.io
